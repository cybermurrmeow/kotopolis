# 🐱 Котополис - Розовый рай для котиков

[![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)](https://www.python.org/)
[![Flask](https://img.shields.io/badge/Flask-2.3.3-red.svg)](https://flask.palletsprojects.com/)

## 🌸 О проекте

**Котополис** - веб-приложение для приюта кошек с милым розовым дизайном.

## ✨ Возможности

- 🐱 Управление котиками
- 🏠 Заявки на усыновление
- 📊 Статистика и графики
- 📧 Email уведомления
- 📁 Экспорт в Excel и PDF

## 🚀 Быстрый старт

```bash
# Клонировать репозиторий
git clone https://github.com/ВАШ_НИК/kotopolis.git
cd kotopolis

# Создать виртуальное окружение
python -m venv venv

# Активировать (Windows)
venv\Scripts\activate

# Установить зависимости
pip install -r requirements.txt

# Запустить приложение
python app.py
